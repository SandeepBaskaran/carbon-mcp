## üîç Finding Specific Features

### Want to add a new tool?

1. Create handler in `src/server/tools/yourTool.ts`
2. Create schema in `src/schemas/yourTool.json`
3. Register in `src/server/toolRegistry.ts`
4. Add tests in `src/server/tools/yourTool.test.ts`
5. Update `README.md` and `examples.md`

### Want to modify logging?

- Edit `src/lib/logger.ts`
- Tests in `src/lib/logger.test.ts`

### Want to change templates?

- Edit files in `src/templates/`
- Template rendering logic in `src/lib/templateUtils.ts`

### Want to add codemods?

- Add to `src/lib/codemodUtils.ts`
- Test in `src/server/tools/codemodReplace.test.ts`
