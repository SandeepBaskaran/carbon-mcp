# Carbon MCP Demo App

A simple demo application showcasing components generated by the carbon-mcp server.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open http://localhost:3000 in your browser

## Using with Carbon MCP

1. Make sure the carbon-mcp server is running:
   ```bash
   cd ../..
   npm run dev
   ```

2. Generate a component:
   ```bash
   curl -X POST http://localhost:4000/tool/generateComponent \
     -H "Content-Type: application/json" \
     -d '{
       "component_name": "MyCard",
       "target_path": "demo/demo-app/src/components/MyCard",
       "dry_run": false,
       "confirm_destructive": true
     }'
   ```

3. Import and use the generated component in `src/App.tsx`:
   ```tsx
   import MyCard from './components/MyCard/MyCard';
   
   // In your component:
   <MyCard>Hello World</MyCard>
   ```
